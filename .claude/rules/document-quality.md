# Document Quality Standards

- Include project name, version, and review date in document headers
- Requirement IDs: `FR-XXX` (functional), `NFR-XXX` (non-functional)
- Finding IDs: `FIND-XXX` format
- Issue IDs: `ISS-XXX` format
- Mermaid diagrams must include alternative text description
- AI proposals must be prefixed with `[AI提案]`
- Each review document must track changes from prior reviews
