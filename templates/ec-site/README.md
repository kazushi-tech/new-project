# EC Site Requirements Template

This template provides a starting structure for e-commerce site requirement documents.

## Usage

1. Copy the template files to your `requirements/` directory
2. Fill in each section with your specific requirements
3. Create a PR on a `feat/requirements-*` branch
4. SpecForge Review will automatically review your draft

## Template Files

- `requirements-template.md` - Main requirements template with EC-specific sections
