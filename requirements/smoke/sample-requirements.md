# サンプル要件定義書（スモークテスト用）

## 機能要件

### FR-001: ユーザー登録

- **優先度**: must
- **説明**: 新規ユーザーがメールアドレスとパスワードでアカウントを作成できる。
- **受入条件**:
  - [ ] メールアドレスの形式を検証する
  - [ ] パスワードは8文字以上
  - [ ] 登録完了後に確認メールを送信する

### FR-002: ログイン

- **優先度**: must
- **説明**: 登録済みユーザーが認証情報でログインできる。

### FR-003: プロフィール編集

- **優先度**: should
- **説明**: ユーザーが自分のプロフィール情報を更新できる。
- **受入条件**:
  - [ ] 表示名を変更できる

## 非機能要件

### NFR-001: レスポンスタイム

- API応答は95パーセンタイルで500ms以内とする。
